ตั้งค่า Cloudinary

เชื่อมต่อกับ Cloudinary เพื่อบันทึกข้อมูลอุปกรณ์ออนไลน์
ทุกคนที่เปิดเว็บจะเห็นข้อมูลเดียวกัน

1 สมัคร Cloudinary (ฟรี)

ไปที่ cloudinary.com → สมัครบัญชีฟรี → ยืนยันอีเมล

2 คัดลอก Cloud Name
เข้า Dashboard ของ Cloudinary → จะเห็น Cloud Name อยู่ด้านบน
เช่น dxxxxxxxx → คัดลอกมาใส่ด้านล่าง
3 สร้าง Upload Preset
ไปที่ Settings (ไอคอนฟันเฟือง) → Upload → เลื่อนลงหา Upload presets

กด Add upload preset → ตั้งค่าดังนี้:
• Upload preset name — ตั้งชื่อเช่น equipment_preset
• Signing Mode — เลือก Unsigned
• Unique filename — ปิด (false)
• Overwrite — เปิด (true)
• Delivery type — upload

แล้วกด Save

อยู่ใน Dashboard หลัก หรือ Settings → Account

ชื่อ preset ที่สร้างในขั้นตอนที่ 3 (ต้องเป็น Unsigned)
